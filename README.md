## Locked product direction

EasyUI Guardian Launcher is a privacy-first, offline-first Android launcher that gives children a safer, simpler, parent-approved phone surface.

It is **not** a surveillance app, not a full parental-control replacement, and not a guaranteed Android lockdown tool.

## Primary promise

> Let your child use a phone without giving them the whole smartphone experience at once.

## Tagline

> A safer home screen for your child’s first phone.

## MVP identity

The MVP should provide:

- default launcher support
- child-safe home screen
- parent PIN
- approved app tiles
- parent contact tile
- emergency tile
- Home / School / Sleep modes
- offline-first setup
- no account
- no backend
- no ads
- no child analytics
- no secret monitoring

## How to use this pack

1. Place this folder in the root of the new repository.
2. Give `prompts/Final_AI_Developer_Prompt.md` to the AI coding agent.
3. The agent must first inspect the repository, then create or update `copilot_session.md`.
4. The agent must implement in phases and produce evidence reports before claiming success.
