# Project Brief — EasyUI Guardian Launcher

## Working name

**EasyUI Guardian Launcher**

Alternative names kept open:

- EasyUI Kids Launcher
- EasyUI Parent Launcher
- KidSafe Home Launcher
- Guardian Launcher

Current preferred name: **EasyUI Guardian Launcher**

## One-line concept

A child-safe Android launcher that helps parents create a calmer, limited, routine-based phone experience for a child’s phone.

## Product thesis

Modern smartphones are too open, distracting, cluttered, and difficult to supervise for children, especially when the child is using a first phone or shared family phone.

Many parents do not need heavy surveillance. They need a safer phone surface where the child sees only what is appropriate:

- parent contact
- emergency contact
- learning apps
- approved games
- camera/photos if allowed
- limited communication apps
- no confusing full app drawer
- no unnecessary clutter
- no easy layout changes

## Product promise

The child gets a phone that feels useful and age-appropriate.  
The parent gets peace of mind without turning the phone into a complicated control system.

## Core positioning

EasyUI Guardian Launcher is a launcher-first child safety layer.

It should complement system tools such as Google Family Link, not replace them.

## What the project is

- A safer home screen for a child’s first phone.
- A parent-configured launcher.
- A routine-based child phone experience.
- A simple offline-first app with minimal permissions.
- A trust-first product.

## What the project is not

- A full parental control suite.
- A surveillance tool.
- A message/call monitoring app.
- A location tracking app.
- A browser history tracker.
- A guaranteed kiosk lockdown solution.
- A replacement for Android system-level parental controls.

## Strategic direction

The product should create **guided independence**, not secret surveillance.

The strongest market angle is:

> Let your child use a phone without giving them the whole smartphone experience at once.
