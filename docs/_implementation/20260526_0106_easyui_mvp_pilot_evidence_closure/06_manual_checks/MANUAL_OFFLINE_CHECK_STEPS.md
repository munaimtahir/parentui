# Manual Offline Check Steps (Airplane Mode)

Device: `08357252AE006901`

Some OEM builds block ADB airplane-mode toggles. Use manual steps:

1. Turn on airplane mode manually.
2. Open EasyUI.
3. Confirm child home works.
4. Open parent dashboard (PIN).
5. Switch Home / School / Sleep.
6. Open Setup Health.
7. Turn airplane mode off.

Record result: PASS / FAIL / NOT DONE and attach screenshots if possible.
