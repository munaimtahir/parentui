# Remaining Manual Device Checks (2026-05-25)

Even with emulator green, a launcher still needs physical-device confirmation for:

- Default launcher selection behavior across OEMs (Samsung/Vivo/Tecno/etc.)
- HOME/RECENTS interactions and OEM restrictions
- Lock screen / keyguard edge cases
- Emergency calling behavior and dialer integration on real hardware
- Installed-app list behavior with real Play-installed apps
