# EasyUI Internal Family Pilot — Setup Guide

This build is for **internal family pilot only** (supervised use). It is **not** a full phone lockdown system.

## What you need (5–15 minutes)
- The phone you’re testing on
- A parent phone number and an emergency number (example: local emergency services)

## Setup steps
1. Install the EasyUI APK on the phone.
2. Open **EasyUI Guardian Launcher**.
3. Complete onboarding:
   - Create a **Parent PIN** (store it somewhere safe).
   - Add a **Parent contact** (name + phone).
   - Confirm **Emergency contact** details.
   - Approve a few apps that your child can use.
4. On the “Setup Completed” screen, review **Default launcher** status.
5. Set EasyUI as the **default Home app**:
   - Tap **Set default home**, then choose **EasyUI** as the Home app.
6. Press the **HOME** button:
   - Confirm it returns to EasyUI.
7. Open **Parent Dashboard**:
   - From child home, tap **Parent setup** (or the parent icon) and enter the PIN.
8. Review **Setup Health**:
   - Fix any items marked “Not set” or “Needs setup”.
9. Review **Ready to hand over?** checklist:
   - Try to get to “Almost ready” or “Ready” before giving the phone to a child.
10. Read **Help & Limitations**:
   - EasyUI is a safer home screen. For deeper restrictions, use Android settings or Google Family Link.

## If HOME still goes to the normal launcher
- Go to Android Settings → Default apps → Home app, and re-select **EasyUI**.
