# Pilot Checklist

## A) Before giving phone to child
- EasyUI installed and opens
- Onboarding complete
- Parent PIN set
- Parent contact added
- Emergency contact added
- At least one app approved
- EasyUI set as default Home app
- Pressing HOME returns to EasyUI
- Setup Health checked
- Ready checklist reviewed
- Limitations understood
- (Optional) Export backup if available

## B) 10-minute parent test
- Parent tile opens dialer (number filled)
- Emergency tile opens dialer (number filled)
- Parent Dashboard requires PIN
- Child cannot access parent screens without PIN
- Child home shows only approved tiles (no full app drawer)

## C) Child handover test (supervised)
- Child can find Parent tile
- Child can find Emergency tile
- Child can open an approved app and return safely
- Child understands mode label (Home/School/Sleep)
- No scary warnings shown to child

## D) Offline test (manual)
Because some devices block ADB airplane-mode toggles:
- Turn on airplane mode manually
- Confirm EasyUI still opens
- Confirm child home renders
- Confirm parent dashboard opens
- Turn airplane mode off
