# Rollback Guide (Return phone to normal launcher)

1. Open **Android Settings**
2. Search for **Default apps**
3. Open **Home app**
4. Select the phone’s original launcher (OEM launcher)
5. (Optional) Open EasyUI and export settings (if backup exists)
6. Uninstall EasyUI if needed

## If you can’t find “Home app”
- In Settings search, try: “Home”, “Default apps”, “Launcher”.

## If parent forgot the PIN
- EasyUI does not have a cloud reset.
- You may need to clear EasyUI app storage in Android Settings to reset local configuration.
