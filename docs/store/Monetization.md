# Monetization

## Recommended model

Freemium with one-time premium unlock.

## Avoid at launch

- ads
- child-targeted monetization
- aggressive upsells
- confusing pricing
- subscription-first model
- dark patterns

## Free tier

Free tier should include:

- basic child launcher
- parent contact
- emergency tile
- limited app selection
- basic parent PIN
- basic child home screen
- one or limited mode setup

## Premium tier

Premium tier can include:

- multiple modes
- mode-specific app assignment
- advanced layout lock
- backup/restore
- multiple child profiles later
- age-based presets
- premium themes
- scheduled modes later

## Pricing hypothesis

Start with simple one-time unlock.

Avoid subscription until the product has cloud features or ongoing service cost.

## Monetization principle

Do not monetize the child directly.

Premium messaging should be parent-facing and practical:

- more routines
- more layouts
- more profiles
- better setup tools
- backup/restore
