# Goals

## MVP goals

- Create a functioning Android launcher.
- Provide a safe child home screen.
- Allow parent-approved apps only inside launcher.
- Add parent PIN protection.
- Add parent and emergency contact tiles.
- Add Home / School / Sleep modes.
- Keep the app offline-first.
- Avoid unnecessary permissions.
- Produce testable, maintainable code.

## V1 goals

- Improve setup experience.
- Add mode-specific app assignment polish.
- Add local backup/restore.
- Add age presets.
- Add paid premium unlock structure.
- Add scheduled modes.
- Improve store readiness.

## Non-goals

- Full parental control suite.
- Cloud monitoring.
- Location tracking.
- Message monitoring.
- Web filtering.
- Call recording.
- Full Android lockdown.
- Device-owner mode in consumer MVP.
