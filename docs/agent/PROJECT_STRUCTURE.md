# Project Structure

Recommended root structure after Android project creation:

```text
easyui-guardian-launcher/
  app/
    src/main/
      java|kotlin/
      res/
      AndroidManifest.xml
  docs/
    product/
    scope/
    ux/
    privacy_compliance/
    technical/
    testing/
    store/
    agent/
    _implementation/
  prompts/
    Final_AI_Developer_Prompt.md
  templates/
    copilot_session_template.md
  .github/
    workflows/
  README.md
  copilot_session.md
```

## Documentation folders

### docs/product

Product definition and positioning.

### docs/scope

MVP, V1, roadmap, risks.

### docs/ux

Child and parent UI concepts.

### docs/privacy_compliance

Guardrails, privacy, permissions, Android limitations.

### docs/technical

Architecture, data model, internal interfaces, setup, CI/CD.

### docs/testing

Test plan and QA checklist.

### docs/store

Play Store and monetization notes.

### docs/agent

Agent workflow, goals, tasks, contribution rules.

### docs/_implementation

Sprint evidence reports generated by AI agents.
