# V1 Scope

## V1 goal

Convert the useful MVP into a polished, parent-friendly paid product.

## V1 feature groups

### Parent Safety Pack

- improved setup wizard
- improved allowed app selection
- mode-specific app lists
- layout lock refinement
- stronger parent dashboard
- Family Link recommendation guide
- local backup/export
- reset and restore layout

### Age presets

Add optional presets:

#### Age 5–8

- parent call
- emergency
- 2–3 learning apps
- camera optional
- no browser
- no social apps
- no full app drawer

#### Age 9–12

- school apps
- approved games
- camera/photos
- selected communication apps
- browser optional
- Home / School / Sleep modes

#### Age 13+

- focus launcher
- study mode
- app grouping
- distraction reduction
- less childish design
- more independence

### Routine Pack

- scheduled School Mode
- scheduled Sleep Mode
- bedtime prompt
- exam mode
- travel mode

### Premium unlock

A one-time premium unlock is preferred.

V1 premium candidates:

- multiple modes
- advanced app assignment per mode
- layout lock
- backup/restore
- multiple child profiles later
- age presets
- premium themes
- scheduled modes

## V1 exclusions

Even in V1, avoid:

- secret monitoring
- ads
- location tracking
- message reading
- call recording
- microphone access
- cloud dependency
- device-owner mode in consumer version

## Separate future track

Stronger restrictions may be explored only as a separate managed-device edition.
